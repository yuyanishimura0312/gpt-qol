## 仮説一覧

### H1: 「対話的ケア」仮説 [確信度: HIGH]
**ロボットケアは「タスク効率化」から「対話と共存の体験」自体が目的化する**

- 転換メカニズム: 目的地化
- 自動車前例: 移動手段→ロードトリップ体験（1930年代）
- エビデンス: PARO（メタアナリシスで抗精神病薬削減確認）、LOVOT（高齢者の前向き行動誘発）、OriHime（障害者の「本物のつながり」報告）
- タイムライン: 2025-2032
- QoL主軸: emotional + relational
- 条件: 長期接触、文化的受容性（日本のアニミズム的ロボット観が有利）

### H2: 「自律回復のシンボル」仮説 [確信度: HIGH]
**外骨格・ロボット義肢は「機能回復」から「自律的アイデンティティ」のシンボルに昇華する**

- 転換メカニズム: シンボル化
- 自動車前例: 輸送手段→個人的自由のシンボル（1920年代）
- エビデンス: 軟体外骨格96.2%精度（Nature Communications）、WalkON歩行努力20%削減
- タイムライン: 2025-2030
- QoL主軸: autonomy + meaning
- レバレッジ: 当事者主導のナラティブ形成が鍵。パラリンピックのメディア効果。

### H3: 「ロボット共創文化」仮説 [確信度: MEDIUM]
**ロボットとの芸術的共創が独自のアートジャンルとして確立する**

- 転換メカニズム: 目的地化 + コミュニティ化
- 自動車前例: 輸送手段→カーデザイン芸術（ピニンファリーナ、1950年代）
- エビデンス: Shimonロボット音楽家（Georgia Tech）、ロボットダンス（MIT/Sydney）
- タイムライン: 2028-2038
- QoL主軸: aesthetic + cultural
- レバレッジ: アーティストの参入と批評言語の成立が転換条件。

### H4: 「ロボット愛好家エコシステム」仮説 [確信度: MEDIUM]
**ロボットオーナーのコミュニティがカーカルチャー同様の愛好家文化を形成する**

- 転換メカニズム: コミュニティ化
- 自動車前例: 個人所有→クラシックカークラブ、チューニング文化
- エビデンス: AIBO愛好家（Sony復刻）、LOVOT飼い主コミュニティ、RoboCup
- タイムライン: 2027-2035
- QoL主軸: relational + meaning
- レバレッジ: 手頃な価格のパーソナルロボットの登場がトリガー。

### H5: 「スローロボティクス」仮説 [確信度: MEDIUM]
**ロボット効率化への抵抗から「丁寧さ」「手作り」「対面」の価値再発見運動が生まれる**

- 転換メカニズム: 抵抗からの創造
- 自動車前例: 公害・孤立→カーフリー運動（1991年〜）
- エビデンス: カント倫理フレームワーク（2025）、依存性・社会孤立リスク文献
- タイムライン: 2030-2040
- QoL主軸: meaning + cultural
- レバレッジ: 雇用喪失の可視化と介護の人間性論争が発火点。

### H6: 「感覚拡張体験」仮説 [確信度: LOW-MEDIUM]
**ハプティクス・触覚技術が「新しい感覚体験のカテゴリ」を創出する**

- 転換メカニズム: 目的地化
- GPT前例: 蓄音機→音楽鑑賞、電気→ネオン夜景美学
- エビデンス: S-CATアフェクティブタッチ（ストレス軽減確認）、触覚代行（視覚障害者QoL+5）
- タイムライン: 2028-2038
- QoL主軸: aesthetic + emotional
- レバレッジ: 「情報伝達」から「美的体験」へのフレーミング転換。

### H7: 「文化的ロボット観の輸出」仮説 [確信度: MEDIUM]
**日本のアニミズム的ロボット観が西洋の「脅威論」を超える受容モデルとして世界に広がる**

- 転換メカニズム: シンボル化 + コミュニティ化
- 自動車前例: 日本の軽自動車文化→グローバルシティカー概念
- エビデンス: ロボット僧侶Mindar、付喪神概念、OriHimeカフェの国際的注目
- タイムライン: 2025-2040
- QoL主軸: cultural + meaning
- レバレッジ: 日本のロボット実践の国際可視化と文化人類学的研究。
